# Fleet

Para execução do projeto é necessario:
1) Instalação local do MongoDB ou redefinição da classe 'br.com.cinq.volvo.fleet.dao.MongoDB' para o path do banco de dados.
2) Instalação local do GlassFish ou outro servidor compativel com distribuição de pacotes WAR em Java
3) Baixar o código fonte do app. (sugestão utilizando a IDE NetBeans)
4) Executar o Build pela IDE. (Neste caso todas as dependencias serão baixadas automaticamente. Projeto realizado com Maven)

OBS. é possível fazer a compilação via linha de comando, para tal será necessario executar o download das dependencias.
Que pode ser feito manualmente substituindo o arquivo Maven ou pela linha de comando Maven.
